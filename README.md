### En:
#### Task 1
Create two functions: one to convert degrees Celsius to degrees Fahrenheit, and another to convert degrees Fahrenheit to degrees Celsius.
- In the first case, the function should take a temperature in Celsius and return its value in Fahrenheit.
Conversion formula:
F = C * 1.8 + 32
- In the second case, the function should take a temperature in Fahrenheit and return its value in Celsius.
Conversion formula:
C = (F - 32) / 1.8
Call each function with different values and display the results in the console.

#### Example 1:

Call: celsiusToFahrenheit(25)
Output: 77

#### Example 2:

Call: fahrenheitToCelsius(77)
Output: 25

### Ru:
#### Задача 2

Создайте две функции для перевода градусов Цельсия в градусы Фаренгейта и, наоборот, градусов Фаренгейта в градусы Цельсия. 

В первом случае функция принимает значение температуры в градусах Цельсия и возвращает его значение в градусах Фаренгейта. Формула для перевода:

F = C * 1.8 + 32

Во втором случае функция принимает значение температуры в градусах Фаренгейта и возвращает его значение в градусах Цельсия. Формула для перевода:

C = (F - 32) / 1.8 

Вызовите каждую функцию с различными значениями и выведите результаты в консоль.

#### Пример 1:

Вызов: celsiusToFahrenheit(25)

Вывод: 77

#### Пример 2:

Вызов: fahrenheitToCelsius(77)

Вывод: 25
